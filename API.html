Api requests

Specific Site Request

Request URLs for thumbnail look up are in the following format:

api.helipaddy.com/sites/N

With an Api-Key XXXXXXXXXX as a header.

Parameters:
N = dec value pad number

api.helipaddy.com/sites/15
